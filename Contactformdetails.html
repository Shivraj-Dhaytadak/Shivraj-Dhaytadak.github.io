Details:

This Project was my first attempt at Database as Backend As I previously known to Web designing.
The Purpose of this project was it can be considered as part of any website to have some sort of way that Visiter can able to report thier feedback or issues faced to admin.
In this Project - their are two Major Section :

  1. User Form
  2. Admin section

Here from user perspective user will only see Contact form and input fieldset and send button
In admin section we have
  • Dashborad - Show new Messages received and user info for reference as well as status as being read or unread message
  • Manage contact : here all last 7 days messages are displayed also total contact made via form
  • Report - All Reported Messages displayed here
  • Update - here admin can change mail and reset password
